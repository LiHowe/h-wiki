## 通过读源码可以学到什么

+ 作者编码习惯及思路
+ 更深入的了解代码并在之后的使用过程中轻车熟路
+ 可能有一些奇技淫巧
+ 整体项目架构
+ 通过看`package.json`来了解一些未曾使用过的库, 并知晓其用途
