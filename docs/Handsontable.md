# HandsonTable

[官网](https://handsontable.com/)



## 配置项

`data`: 表格数据



`rowHeaders`: 是否显示行头(序号行)

`colHeaders`: 是否显示列头(序号列)



`licenseKey`: 授权码, 开源项目可以使用`'non-commercial-and-evaluation'`



用于显示空白行/列

> 表格行数据必须是数组, 不能是对象

+ `minRows`: 设置最小行数

+ `minCols`: 设置最小列数
