# H-Wiki

使用 `VuePress2.x` 构建, 重构于上一个未完成的博客(使用`nuxt`构建).

将会通过`netlify`进行部署, 配合阿里DNS解析

## 目录结构

```text

|- .obsidian        <- 本地markdown编辑器Obsidian配置
|- .script          <- 打包脚本等
|- .vuepress        <- vuepress 相关配置
|- .yarn            <- yarn3配置
|- content          <- Markdown内容

```
